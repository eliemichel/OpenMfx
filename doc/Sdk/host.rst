Host SDK
========

The Host SDK is a C++ library that aims at making the support of OpenMfx plugins in your application easier than using the raw OpenMfx API. For the reference of the underlying C API, see :ref:`ApiReference`.

When using CMake, link your project to the target ``OpenMfx::Sdk::Cpp::Host`` to use this SDK.
