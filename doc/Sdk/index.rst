SDK Reference
=============

The SDK is a C++ library that aims at making the development of OpenMFx plugins and host easier than using the raw OpenMfx API. For the reference of the underlying C API, see :ref:`ApiReference`.

The SDK is different whether you want to implement a plugin or add the support for running plugins to an existing host applications.
